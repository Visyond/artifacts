Detailed Description

specifies the standard model of a UnoControlContainer.

Exported Interfaces

service com::sun::star::awt::UnoControlModel

Exported Interfaces

interface com::sun::star::container::XIndexContainer

The control serves as an indexed container typically for RoadmapItems as specified in com::sun::star::awt:RoadmapItem.

The RoadmapItems are held in a sequence. When inserting such items their ID is set equal to the Index of their insertion by default. After removing items the CurrentItem property is - when beyond the upper sequence boundaries - set equal to last index of the RoadmapItem Array.

Member Data Documentation

long BackgroundColor
property

specifies the background color (RGB) of the control.

The Default value is white

short Border
property

specifies the border style of the control. 

0: No border
1: 3D border
2: simple border
boolean Complete
property

determines whether the control container is complete or not.

If it is false than a non - interactive RoadmapItem is appended

short CurrentItemID
property

refers to the ID of the currently selected item.

Initially this property is set to "-1" which is equal to "undefined" If the Roadmap Item that the CurrentItemID refers to is removed the property "CurrentItemID" is set to -1

com::sun::star::graphic::XGraphic Graphic
optionalpropertytransient

specifies a graphic to be displayed on the control

If this property is present, it interacts with the ImageURL in the following way:

  • If ImageURL is set, Graphic will be reset to an object as loaded from the given image URL, or NULL if ImageURL does not point to a valid image file.
  • If Graphic is set, ImageURL will be reset to an empty string.

Detailed Description

Macro expander expanding using rtl bootstrap notation.

For details, have a look at http://udk.openoffice.org/common/man/concept/micro_deployment.html#misc

The service can be instantiated via arguments giving an ini/rc file url. Otherwise it will read from a uno.ini/unorc file next to the cppuhelper library.

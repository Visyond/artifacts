Detailed Description

represents absolute “vnd.sun.star.expand” URL references.

Since
OOo 2.3

Exported Interfaces

interface XUriReference

represents the generic features of the URL reference.

The behaviour of the methods of this interface will always reflect the fact that the represented URI reference is an absolute “vnd.sun.star.expand” URL reference.

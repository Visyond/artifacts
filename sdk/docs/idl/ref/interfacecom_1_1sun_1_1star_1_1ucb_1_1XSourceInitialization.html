provides the initialization of a component with any source object.

The service description has to specify which type of interface must be set as parameter.

Hopefully you will only use this, when Source is an com::sun::star::lang::XComponent and this is an com::sun::star::lang::XEventListener. Then you should call com::sun::star::lang::XComponent::addEventListener() from inside the implementation of this method.
